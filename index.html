  • Display Property

      flex - block level flex container
      inline - displayed inline no height and width
      inline-block - displayed inline has height and width
      block - displayed on its own line has height and width
      list-item - display like a li

  • Position Property

      static - normal flow
      fixed - with respect to window, other elements ignore it
      relative - with respect to rendering position
      absolute - with respect to last relative position
      then set left right top bottom

  • Flex Box

    flex-direction - determines main axis
      row ↔ main axis
      column ↕

    justify-content: (along main axis)
      space between (widest)
      space around (mid)
      space evenly (closeset)
      center
      flex-start/end

    align-items: (along cross axis)
      flex-start
      felx-end
      center

    align-self: (applies to individual item)
      flex-start
      felx-end
      center

    flex-wrap: wrap;
